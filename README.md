# 🛒 Zepto SQL Data Analysis Project

## 📌 Overview

This project performs SQL-based data analysis on a Zepto dataset to generate insights related to pricing, inventory, and product performance.

---

## 📂 Dataset

The dataset includes:

* Product name
* Category
* MRP & discounted price
* Quantity & stock availability
* Product weight

---

## 🧠 Approach

* Performed data cleaning (removed invalid records, converted paise to rupees)
* Used aggregation (SUM, AVG) for business insights
* Applied CASE statements for product segmentation
* Created derived metrics like price per gram and revenue

---

## 🔍 Key Analysis

* Top discounted products
* Out-of-stock high-value items
* Revenue per category
* Price per gram analysis
* Inventory weight distribution

---

## 📊 Key Insights

* Bulk products offer better price efficiency
* High MRP items going out of stock may lead to revenue loss
* Inventory distribution varies significantly across categories

---

* MySQL Workbench
* SQL
